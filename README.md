# Expense Tracker (Python OOP Project)

This is a simple Expense Tracker project built using Python and Object-Oriented Programming concepts.

Features:
- Add expenses with category
- View all expenses
- Calculate total expenses
- Save expenses to a file
- Exception handling included

Concepts Used:
- Classes & Objects
- File Handling
- Exception Handling
- Lists & Dictionaries
- Menu Driven Program

Created as part of my Python learning journey.
